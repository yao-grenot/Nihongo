# 第一课

### 单词

わたし	我	

あなた	你、您

あのひと（あの人）	他、地、那介人

あのかた（あの方）	「あのひと」更礼貌的说法



～さん

～ちゃん

～じん	～人	（表示国籍的接尾词，例如：アメリカじん）

せんせい	先生

きようし	教師

がくせい	学生

かいしゃいん	会社員 

しゃいん	社員 （和公司名一起使用，例如：IMCのしゃいん）

ぎんこういん	 銀行員

いしゃ	医者

けんきゅうしゃ 	研究者



だいがく	大学

びょういん	病院



だれ	谁（哪位）

どなた  	（「だれ」的礼貌说法）

～さい	～歲

なんさい	何歳

おいくつ	是「なんさい」的礼貌説法



はい	是、対

いいえ	不、不是



おはよう　ございます	早上好

はじめ（初め）ました	初次见面

ちゆうごくから　来（き）ました	是从中国来的

どうぞ　よろしく お願い（ねがい）します	请多关照（自我介绍时最后讲的话）

失礼（しつれい）ですが	对不起，请问

お名前（なまえ）わ？	您贵姓？

こちらは　～さんです。	这位是～
